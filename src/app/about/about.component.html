About EnergyHub v1.0

A comprehensive IoT monitoring solution designed for real-time demand analysis and industrial energy efficiency.

The Objective

EnergyHub was developed to solve the "visibility gap" in industrial settings. By aggregating telemetry from multiple meters across various production lines, the system provides operators with instant insight into Total Load and Peak Demand.

The Technology

  • Backend: Spring Boot, JPA, PostgreSQL
  • Frontend: Angular 17+, Tailwind CSS
  • Analytics: Chart.js with DTO Aggregation
  • Security: Role-Based Access (Admin/Operator)

How it Works

01. Telemetry

Meters push kWh readings every 30 mins.

➜
02. Aggregation

Java backend groups data into DTOs.

➜
03. Visualization

Angular renders real-time performance charts.
